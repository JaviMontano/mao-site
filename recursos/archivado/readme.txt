# MetodologIA Resource Kit (Recursos) - Documentation

## 1. Strategic Purpose
Provides the "Operational Fuel" for the MetodologIA ecosystem. It offers a clear path from free sampling to premium adoption through an isomorphic resource structure.

## 2. Content Structure
The kit is organized into 12 main categories, each existing in both a FREE (Public) and PREMIUM (Gated/Paid) version:
- Playbooks & E-books: Documentation and strategy.
- Asistentes GPT & Gems: Configuration for native platform tools.
- Automatizaciones (n8n): Workflow templates.
- Mini Apps & Prototipos: Interactive components (Claude, AI Studio, Stitch, v0, Lovable).
- Flujos de Agentes: Advanced autonomous behaviors (Manus, GenSpark).

## 3. Isomorphic Logic
- /recursos/: Landing pages for the FREE universe.
- /recursos/premium/: Deep-dive versions of the same assets.

## 4. Maintenance
Ensure that every new asset added to the /recursos/ subdirectories has a corresponding entry in both the FREE and PREMIUM hubs to maintain the isomorphic experience.
